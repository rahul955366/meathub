# MeatHub Landing Page

Simple, trustworthy landing page designed for Acquire.com listing.

## Purpose

This page is designed for:
- Startup buyers on Acquire.com
- Founders and operators evaluating the business
- CTOs assessing the technical foundation

Not for mass consumer marketing.

## Design Philosophy

- **Simple**: No flashy animations or hype
- **Trustworthy**: Honest about development stage
- **Professional**: Clean typography and layout
- **Calm**: Warm colors, plenty of whitespace

## Key Principles

✅ Clear language  
✅ Honest about being in beta  
✅ Explains the business model simply  
✅ Shows technical credibility  
❌ No fake testimonials  
❌ No revenue claims  
❌ No exaggerated promises  

## Deployment

### Quick Deploy to Vercel

1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Upload the `landing-page` folder
4. Done! Your landing page is live

### Manual Deploy

```bash
# The page is a single HTML file
# Upload index.html to any static host:
# - Vercel
# - Netlify
# - GitHub Pages
# - Any web server
```

## Customization

To customize before deployment:

1. **Email**: Change `founder@meathub.app` to your actual email
2. **Year**: Update `© 2024` if needed
3. **Content**: Adjust text to match current development status

## Tone & Messaging

The page uses:
- Foundational language ("built with")
- Honest stage indicators ("in development")
- Technical credibility (mentions architecture)
- Clear value proposition (hyperlocal, trust)

Avoids:
- Hype words ("revolutionary", "disrupting")
- Fake claims
- Overpromising
